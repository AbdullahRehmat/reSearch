# reSearch

reSearch is a lightweight, fast BM25 based Search Engine written in Python & Go.



### Installation

To run with [Docker-Compose](https://docs.docker.com/compose/) execute the following:

> ./start.sh

View the Web Interface at:

> http://localhost:80/
